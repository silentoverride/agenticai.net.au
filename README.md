# Agentic AI Website

Website project for Agentic AI.

## Direction

Agentic AI presents practical AI implementation services for businesses, including:

- AI agent development
- Workflow automation
- AI strategy and consulting
- Custom integrations
- Internal tools and dashboards
- Training and enablement
- Mobile proxy infrastructure hosting

The site should include a polished services section for mobile proxy infrastructure hosting with a Three.js globe visual: continent silhouettes, city nodes, and animated city-to-city connection streams.
